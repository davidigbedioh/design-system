# Design System (JavaScript)

A portfolio-ready design system built with **React + Vite** using **plain JavaScript** (no TypeScript).

## Includes
- Tokens via CSS variables (light + dark themes)
- Button (variants + sizes)
- Card (header/body)
- Badge (tones)
- Input (label, hint, error + aria-describedby)
- Switch (theme toggle)
- Tabs
- Modal (Escape + click-outside close)
- Toast notifications (aria-live, capped stack)

## Run
```bash
npm install
npm run dev
```
